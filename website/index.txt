h1. autoreload, you don't have to restart your program.

h2. What

'autoreload' reload libraries when the files are updated.

h2. Installing

<pre syntax="ruby">sudo gem install autoreload</pre>

h2. The Basic

<pre syntax="ruby">
# foo.rb
def foo
  1
end
</pre>

<pre syntax="ruby">
# sample1.rb
require 'autoreload'
require 'foo'

autoreload(1, true, 'sample1')

loop {
  puts foo
  sleep 1
}
</pre>

In this situation, run the script 'sample1.rb'.
You see that the number 1 is shown repeatedly.
Then, update the file 'foo.rb'.
For example, change the number 1 to 2 and save it.
Then, the library automatically check the update and reload it.
You'll see the number 2 is shown.
